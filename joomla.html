Joomla!

O que é Joomla? O que é JPlatform?

Joomla!® (pronuncia-se djumla) é um sistemas CMS (do inglês Content Management System, Sistema de Gestão de Conteúdo) gratúito, escrito em PHP, com suporte a múltiplos bancos. O nome também está associado ao JPlatform, um framework PHP voltado a desenvolvimento de aplicações Web que vão além de sites.

Download do CMS Joomla!

Visite e baixe diretamente da página oficial de download do CMS Joomla. Recomento a versões 3.x, porém 2.5.x podem ser usadas sem problemas. Cuidado! Não mantenha com versões obsoletas e inseguras como 1.5, 1.6 e 1.7!

Referências oficiais Joomla!

Se você é um usuário final ou administrador Joomla, recomendo:

  • The Joomla! Extensions Directory (JED), para download de extensões gratúitas e pagas
  • Fórum de suporte oficial, idioma português. Obtenha suporte gratúito, e retribua ajuda.
  • Documentação oficial Joomla!. Wiki com manuais de uso. Não a ignore
  • Joomla! Issue Tracker, para reportar possíveis erros com o CMS
  • Joomla Idea Pool, para sugerir novas características ao CMS

Se além de usuário, você é usuário avançado ou programador, considere:

  • Joomla! Developer Network
  • The Joomla!® User Experience Portal. Portal para discussões de experiência do usuário
  • Joomla Platform Manual. Manual de uso do JPlatform em formato "amigável".
  • Lista de email "Joomla CMS Developer", para assuntos de desenvolvimento com o CMS
  • Lista de email "Joomla CMS Platform", para assuntos de desenvolvimento com o JPlatform
  • Lista de email "Joomla Bug Squad", para discussões do JBS Team.
