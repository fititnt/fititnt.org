JoomlaFox!

JoomlaFox! - Joomla para Firebug Extensão do Firefox

Escrito por Emerson Rocha Luiz
Detalhes
Categoria: Joomla!
Última atualização: 31 Março 2013
Criado: 19 Outubro 2009
Twitter
For one english version, please visit JoomlaFox in english page

O que é?

JoomlaFox são duas coisas diferentes: uma é uma extensão que roda no Firefox (baseada no Drupal for Joomla do Matt Cheney) e a outra é o componente junto ao plugin que são instalados ao Joomla. Se você tiver ambos habilitados, e autenticado como superadministrador, verá algumas informações adicionais sobre seu site Joomla ou de extensões de terceiros.


Nota: esta imagem pode estar desatualizada em relação a versão mais recente do JoomlaFox! Para ter certeza compare com as versões na área de downloads

Para quem é?

JoomlaFox, por enquanto é focado para desenvolvedores Joomla. Você pode testar e usar caso não seja um, mas talvez não vai achar tão util. Talvez no futuro se implemente algo mais interessante para usuários comuns

Como instalar

Installe a versão mais recente do JoomlaFox Joomla Componente Joomla .

Installe a versão mais recente do JoomlaFox Plugin Joomla, e tenha certeza de deixar publicado o plugin 'JoomlaFox'.

Installe a lversão mais recente do JoomlaFox para Firebug-Firefox Extension , e abra a aba 'JoomlaFox' eu sua extensão 'Firebug' do seu Firefox.

Observação: por razões de segurança, você deve estar autenticado como superadministrador para usar esta extensão;

Pré-requisito

Joomla 1.5 como CMS e Firefox 3+ com a extensão Firebug instalada como browser.

Qual é a versão atual?

Por enquanto apenas algumas versões betas estão sendo liberadas, com o intuido de esperar sugestões ( como as que você gostaria ) para que sejam implementadas.
